# P5

![Badge Status](https://ci-as-a-service)

Utility class files for Processing

## Description

Processingまわりの便利クラスをまとめています。

***DEMO:***

それぞれのクラスファイルを参照。

## Features

- HttpUtil
HTTP通信するときの便利クラス

- OscUtil
OSC通信するときの便利クラス

- SerialUtil
Arduinoシリアルポートとかを自動認識して繋いでくれる
.app化した時とかちょっと便利

## Requirement

- Processing 3.0

## Usage & Installation

使用したいP5の.pdeファイルと一緒に「該当クラス.pde」を追加すればok。
